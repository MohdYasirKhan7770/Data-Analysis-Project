## 🚀 Live Demo

Open `index.html` in your browser or run the dev server:

```bash
npm run dev
```

This will open the site at `http://localhost:3000` with live reload.

## ✨ Features

- 🌗 **Dark / Light Theme** — Toggle with button or press `T`
- 📊 **Interactive Dashboard** — Line chart, donut chart, KPI cards
- 📋 **Sortable Data Table** — Search, sort, paginate analysis results
- 🔍 **Feature Filters** — Filter features by category
- 💻 **Code Preview Tabs** — Basic / Advanced / ML snippets with copy button
- 📬 **Contact Form** — Full validation with success feedback
- 📰 **Newsletter Signup** — Email validation
- ⬆️ **Back to Top** — Floating button
- 🎨 **Particle Background** — Animated canvas
- ⌨️ **Keyboard Shortcuts** — `Ctrl+K` search, `T` theme, `Esc` close menu

## 🛠️ Tech Stack

## 🏃 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/prismlytics.git
cd prismlytics

# Start dev server with live reload
npm run dev
```

## 🌐 Deployment

### GitHub Pages
1. Push to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site is live at `https://YOUR_USERNAME.github.io/prismlytics/`

### Netlify (Drag & Drop)
1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag your project folder
3. Done! Instant deployment

### Vercel
```bash
npx -y vercel
```
